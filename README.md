# ClientRegistration

A simple client registration system.
Functionality:
1) Add a client
2) Edit a client
3) Remove a client
4) List all clients
5) Import a .csv file.

Note: a .csv can be viewed here:
https://github.com/MatasSilinskas/ClientRegistration/blob/master/client_examples.csv

# How to run:

```bash
  $ git clone <project>
  $ cd path/to/<project>
  $ php Console.php 
```

# How to run tests:
```bash
  $ ./vendor/bin/phpunit Tests/
```
